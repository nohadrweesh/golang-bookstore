# Description
- This Application is from this [tutorial](https://www.youtube.com/watch?v=1E_YycpCsXw)
- This is a CRUD app written in GoLang , it has apis for a bookstore to create/update/delete books
## How to run the app:
- Clone the repo
- Cd into the app
- Run these command 
  ```
  gi get
  go run main.go
  ```
- The app has 4 routes:
    - GET       /book
    - GET       /book/{id}
    - POST      /book/{id}
    - PUT       /book/{id}/{id}
    - DELETE    /book/{id}/{id}
